# 交易所符文属性价格导出
 
花3个小时撸出来的，做的非常暴力，仅供参考

## 大体的流程：
先手动在交易所选择符文，把传说级和古代级去掉，暂时只能看史诗级的符文
按价格倒排
在trader.py里设置交易所页数（右下角的数字/6+1）
运行trader.py（需要管理员权限）
页面切回龙之谷页面 会有5秒的预留时间
脚本开始自动截图，期间不要移动鼠标操作键盘
截图完毕自动拼接保存图片和时间
图片和时间通过QQ机器人发送给群友
